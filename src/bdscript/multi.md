# $multi
Muliplies the provided numbers.

## Usage
```
$multi[number;..]
```

### Parameters 
- `number` `(Type: Float, Integer || Flag: Required)`: The numbers to multiply. Separate numbers using `;`.

## Example
```
$nomention
$argsCheck[>2;❌ Invalid usage. Usage: `!multiply (number1) (number2)`]
$multi[$message[1];$message[2]]
```
