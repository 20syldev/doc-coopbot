# $randomString
Generates a random combination of letters/numbers.

## Usage
```
$randomString[length]
```

### Parameters
- `length` `(Type: Integer || Flag: Required)`: How long the string should be (max `10`).

## Example
```
$nomention
String: `$randomString[$message]`
```
